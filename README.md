# Tech Portfolio

## Installation

1. **Clone the repository**
   ```
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```

2. **Install dependencies**
   ```
   npm install
   ```

## Run the Project

**Development Mode**
```
npm run dev
```

Open your browser at `http://localhost:5173`

**Build for Production**
```
npm run build
```

**Preview Production Build**
```
npm run preview
```
